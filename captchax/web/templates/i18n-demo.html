🎛️ Interactive Language Switcher

Click the floating language button in the top-right corner to switch languages. The switcher supports:

  • Click outside to close
  • Keyboard navigation (Escape to close)
  • Automatic RTL layout adjustment
  • Persistent selection via localStorage

💻 JavaScript API Usage

Example code for programmatic language switching:

// Initialize i18n await CaptchaXI18n.init({ defaultLocale: 'en' }); // Get translations const title = CaptchaXI18n.t('captcha.title'); const timeText = CaptchaXI18n.t('time.minutesAgo', { n: 5 }); // Switch language await CaptchaXI18n.setLocale('ja'); // Listen for changes CaptchaXI18n.onLocaleChange((locale) => { console.log('Language changed to:', locale); });
