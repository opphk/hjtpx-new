CaptchaX SDK 使用示例

主题切换

1. 基础使用

点击按钮显示滑块验证码

2. Promise 方式

CaptchaX.verify({
    type: 'slider',
    container: '#captcha-container'
}).then(function(result) {
    console.log('验证成功:', result);
}).catch(function(error) {
    console.log('验证失败:', error);
});

3. 回调方式

CaptchaX.onSuccess(function(result) {
    console.log('验证成功:', result.token);
});

CaptchaX.onError(function(error) {
    console.log('验证失败:', error.error);
});

CaptchaX.verify({ type: 'click' });

4. 嵌入指定容器

点击按钮后验证码将显示在这里


5. 自定义配置

CaptchaX.init({
    serverUrl: 'https://your-captchax-server.com',
    appId: 'your-app-id',
    timeout: 10000
});

CaptchaX.verify({
    type: 'slider',
    theme: 'dark',
    autoClose: true
});
