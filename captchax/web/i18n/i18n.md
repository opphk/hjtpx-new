# CaptchaX 多语言支持文档

## 概述

CaptchaX 支持多种语言的本地化，本文档介绍了可用的语言文件和如何添加新语言。

## 支持的语言

### 现有语言

| 语言代码 | 语言名称 | 文件 | 方向 |
|---------|---------|------|------|
| zh-CN | 简体中文 | zh-CN.json | ltr |
| zh-TW | 繁体中文 | zh-TW.json | ltr |
| en | English | en.json | ltr |
| ja | 日本語 | ja.json | ltr |
| ko | 한국어 | ko.json | ltr |
| ar | العربية | ar.json | rtl |

### 新增语言

| 语言代码 | 语言名称 | 文件 | 方向 |
|---------|---------|------|------|
| fr | Français | fr.json | ltr |
| de | Deutsch | de.json | ltr |
| es | Español | es.json | ltr |
| ru | Русский | ru.json | ltr |

## 文件结构

每个语言文件都是 JSON 格式，包含以下主要部分：

- `code`: 语言代码
- `name`: 语言名称（本地语言）
- `dir`: 文本方向（ltr 左到右，rtl 右到左）
- `common`: 通用词汇
- `captcha`: 验证码相关文本
- `admin`: 管理后台文本
- `messages`: 提示消息
- `time`: 时间相关文本

## 添加新语言

1. 复制 `en.json` 为新语言文件，命名为 `<语言代码>.json`
2. 修改 `code` 和 `name` 字段
3. 翻译所有文本内容
4. 确保文本方向（`dir`）正确设置
5. 在应用中注册新语言

## 更新翻译

如需更新翻译，请：
1. 找到对应语言文件
2. 修改相应的键值
3. 保持 JSON 格式正确
4. 提交更改
