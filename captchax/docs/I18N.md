# CaptchaX 国际化 (i18n) 文档

## 概述

CaptchaX 支持多语言国际化，系统内置 6 种语言支持：

| 语言 | 代码 | 方向 | 状态 |
|------|------|------|------|
| 简体中文 | `zh-CN` | LTR | ✅ 已实现 |
| 繁体中文 | `zh-TW` | LTR | ✅ 已实现 |
| 英语 | `en` | LTR | ✅ 已实现 |
| 日语 | `ja` | LTR | ✅ 已实现 |
| 韩语 | `ko` | LTR | ✅ 已实现 |
| 阿拉伯语 | `ar` | RTL | ✅ 已实现 |

## 目录结构

```
web/
├── i18n/
│   ├── zh-CN.json      # 简体中文
│   ├── zh-TW.json      # 繁体中文
│   ├── en.json         # 英语
│   ├── ja.json         # 日语
│   ├── ko.json         # 韩语
│   └── ar.json         # 阿拉伯语
└── static/
    ├── i18n.js              # 国际化核心库
    ├── i18n-integration.js  # 集成辅助库
    └── language-switcher.js # 语言切换器
```

## 快速开始

### 1. 初始化国际化

```html
<script src="/static/i18n.js"></script>
<script src="/static/language-switcher.js"></script>
<script src="/static/i18n-integration.js"></script>

<script>
    CaptchaXI18n.init({
        defaultLocale: 'en',
        storageKey: 'captchax_locale'
    }).then(function() {
        // 初始化完成
        LanguageSwitcher.init({
            position: 'top-right'
        });
    });
</script>
```

### 2. 在 HTML 中使用翻译

```html
<!-- 文本翻译 -->
<span data-i18n="captcha.title">安全验证</span>

<!-- 属性翻译 -->
<button data-i18n-attr='{"title": "admin.logout", "aria-label": "admin.logout"}'>退出</button>
```

### 3. 在 JavaScript 中使用翻译

```javascript
// 获取翻译文本
var title = CaptchaXI18n.t('captcha.title');
var loadingText = CaptchaXI18n.t('captcha.loading');

// 带参数的翻译
var timeAgo = CaptchaXI18n.t('time.minutesAgo', { n: 5 }); // "5分钟前"
```

## API 参考

### I18n 全局对象

#### `I18n.init(options)`

初始化国际化系统。

```javascript
await I18n.init({
    defaultLocale: 'en',      // 默认语言
    localeParam: 'lang',       // URL 参数名称
    storageKey: 'captchax_locale' // localStorage 键名
});
```

#### `I18n.setLocale(locale)`

设置当前语言。

```javascript
await I18n.setLocale('ja');  // 切换到日语
```

#### `I18n.t(key, params)`

获取翻译文本。

```javascript
I18n.t('admin.dashboard');                    // "仪表盘"
I18n.t('time.hoursAgo', { n: 3 });           // "3小时前"
```

#### `I18n.getCurrentLocale()`

获取当前语言代码。

```javascript
var current = I18n.getCurrentLocale();  // "zh-CN"
```

#### `I18n.onLocaleChange(callback)`

监听语言切换事件。

```javascript
I18n.onLocaleChange(function(newLocale, oldLocale) {
    console.log('语言从', oldLocale, '切换到', newLocale);
    // 重新渲染页面内容
});
```

#### `I18n.getSupportedLocales()`

获取支持的语言列表。

```javascript
var locales = I18n.getSupportedLocales();
// ['zh-CN', 'zh-TW', 'en', 'ja', 'ko', 'ar']
```

#### `I18n.getLocaleInfo(locale)`

获取语言信息。

```javascript
var info = I18n.getLocaleInfo('ar');
// { code: 'ar', name: 'العربية', dir: 'rtl' }
```

### LanguageSwitcher 语言切换器

#### `LanguageSwitcher.init(options)`

初始化语言切换器。

```javascript
LanguageSwitcher.init({
    position: 'top-right',  // 位置: top-right, top-left, bottom-right, bottom-left
    showFlag: true,         // 是否显示国旗图标
    compact: true,          // 是否使用紧凑模式
    onChange: function(locale) {
        console.log('语言已切换到', locale);
    }
});
```

## 语言自动检测

系统按以下优先级自动检测用户语言：

1. **localStorage 存储的语言** - 用户上次选择
2. **URL 参数** - `?lang=ja`
3. **Accept-Language 请求头** - 浏览器语言设置
4. **HTML lang 属性** - `<html lang="zh-CN">`
5. **默认语言** - 配置的 fallback 语言

### 检测示例

```javascript
// Accept-Language: zh-CN,zh;q=0.9,en;q=0.8,ja;q=0.7
// 检测结果: zh-CN

// Accept-Language: ja-JP,en-US;q=0.9
// 检测结果: ja

// Accept-Language: ar-SA,ar;q=0.9
// 检测结果: ar (RTL 布局自动启用)
```

## RTL 支持

阿拉伯语 (`ar`) 启用 RTL (从右到左) 布局：

- 页面方向自动切换为 `rtl`
- CSS 自动应用 RTL 样式
- 切换器位置自动调整

```javascript
// RTL 语言列表
var rtlLocales = ['ar', 'he', 'fa'];
```

## 添加新语言

1. 在 `web/i18n/` 目录创建新的翻译文件（如 `de.json`）

2. 在 `I18n.supportedLocales` 数组中添加新语言代码

3. 在 `I18n.localeMap` 中添加语言映射

```javascript
// 在 i18n.js 中添加
supportedLocales: ['zh-CN', 'zh-TW', 'en', 'ja', 'ko', 'ar', 'de'],
localeMap: {
    // ... 现有映射
    'de': 'de',
    'de-DE': 'de'
}
```

4. 在 `language-switcher.js` 的 `getFlagEmoji` 方法中添加国旗 emoji

## 最佳实践

### 1. 使用 data-i18n 属性

```html
<!-- 推荐 -->
<h1 data-i18n="admin.title">管理后台</h1>

<!-- 不推荐 -->
<h1 id="title">${t('admin.title')}</h1>
```

### 2. 包含占位符参数

```json
// 翻译文件
"time.minutesAgo": "{n}分钟前"

// 使用
CaptchaXI18n.t('time.minutesAgo', { n: 5 }); // "5分钟前"
```

### 3. RTL 测试

使用阿拉伯语测试时，确保：
- 所有 flexbox 布局方向正确
- 文字对齐方式正确
- 切换器位置正确

### 4. 加载策略

```javascript
// 预加载所有翻译（推荐）
await CaptchaXI18n.init();

// 按需加载
await I18n.loadTranslation('fr');
await I18n.setLocale('fr');
```

## 示例代码

### 完整集成示例

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title data-i18n="admin.title">CaptchaX Admin</title>
</head>
<body>
    <div id="app">
        <h1 data-i18n="admin.dashboard">Dashboard</h1>
        <p data-i18n="captcha.loading">Loading...</p>
    </div>

    <script src="/static/i18n.js"></script>
    <script src="/static/language-switcher.js"></script>
    <script src="/static/i18n-integration.js"></script>
    <script>
        document.addEventListener('DOMContentLoaded', async function() {
            await CaptchaXI18n.init({
                defaultLocale: 'en'
            });

            LanguageSwitcher.init({
                position: 'top-right',
                onChange: function(locale) {
                    console.log('Language changed to:', locale);
                }
            });
        });
    </script>
</body>
</html>
```

### React/Vue 集成

```javascript
// React Hook 示例
function useI18n() {
    const [locale, setLocale] = useState(CaptchaXI18n.getLocale());

    useEffect(function() {
        CaptchaXI18n.onLocaleChange(function(newLocale) {
            setLocale(newLocale);
        });
    }, []);

    return {
        t: CaptchaXI18n.t,
        locale: locale,
        setLocale: CaptchaXI18n.setLocale
    };
}
```

## 故障排除

### 翻译未加载

```javascript
// 检查翻译是否加载
console.log(I18n.translations['zh-CN']);
console.log(I18n.getCurrentLocale());
```

### 语言切换不生效

```javascript
// 确保调用 updateDOM
I18n.updateDOM();

// 或手动触发更新
CaptchaXI18n.applyTranslations();
```

### RTL 布局异常

```javascript
// 手动设置方向
document.documentElement.setAttribute('dir', 'rtl');
document.documentElement.setAttribute('lang', 'ar');
```
